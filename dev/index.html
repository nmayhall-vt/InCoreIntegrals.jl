InCoreIntegrals.subset — Method
subset(ints::InCoreInts, list; rmd1a, rdm1b)

Extract a subset of integrals acting on orbitals in list, returned as InCoreInts type and contract a 1rdm to give effectve 1 body interaction

Arguments

  • ints::InCoreInts: Integrals for full system
  • list: list of orbital indices in subset
  • rdm1a: 1RDM for embedding α density to make CASCI hamiltonian
  • rdm1b: 1RDM for embedding β density to make CASCI hamiltonian

InCoreIntegrals.subset — Method
subset(ints::InCoreInts, list)

Extract a subset of integrals acting on orbitals in list, returned as InCoreInts type. Because a constant is necessarily a global quantity, we return an InCoreInts object with h0 = 0.0.

Arguments

  • ints::InCoreInts: Integrals for full system
  • list: list of orbital indices in subset
